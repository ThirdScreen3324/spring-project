Welcome to a compact guide to linux distros

Picking a linux distribution is usually the first step in learning linux. For some, it's quite easy but for others it can be quite hard. This website is meant as a small starter guide which includes the basics that may point you in the right direction.

  • Debian

    A stable and beginner friendly distro with a simple installation process

  • Arch

    A cutting-edge distro that updates frequently but might lead to system breakage. Manual installation

  • Gentoo

    DIY distro, for advanced users that want to control everything in their system
