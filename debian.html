Welcome to a summery of the Debian distro

Debian is known for its focus on stability, security, and reliability. It uses thoroughly tested software packages and receives long-term support, making it a popular choice for servers, development environments, and everyday computing. Software is installed and managed through the APT package manager*, which provides access to a large collection of applications.

*Links to a website related to the ubuntu distro, not debian, but they use the same package manager so it's still relevent
